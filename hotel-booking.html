Hotel Finder

Full-stack web application built with Python and Flask.

User Registration

Users are able to create an account to sign in and view the other pages that are only accessible if the user is authenticated.

When users sign up the will be automatically sent to their profile page and from there they can view the hotels and rooms pages. When a user registers, the data gets saved into a database and the user can then use that data to log back in.

Hotels

The hotels page displays different hotels from different areas around the U.S.

Each hotel has an image as well as the name of the hotel and the location. Users can click on any of the hotels and they will be sent to the rooms page to view the available rooms.

Rooms

The rooms page contains all of the rooms that are available.

Each hotel has about 2-3 rooms associated with them. Every room has a room-type as well as it's price, availability, and the hotel it belongs to. If a room is unavailable, the user will not be able to click the 'book room' button.

Booking

The booking page allows the user to book the room of their choosing.

When the user picks a room, they get sent to the booking page for that specific room and are asked to fill out the dates they wish to stay. After successfully booking, it will get added to their profile page.

Profile

The profile page will keep track of any current bookings the user has.

This page displays all of the user's bookings and allows them to cancel any bookings if needed.
