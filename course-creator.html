Course Creator

CRUD app built with .NET that allows the user to update a course catalog for a school.

Back to Projects

Adding New Course

Users are able to click on the 'Add Course' button which brings them to this page where they fill out a form with information about the course.

Each course has a subject, title, description, and credit hours. There are validation checks in place to make sure the user enters valid data. When the form is filled out and the user clicks 'Add Course', the new course will be added to the catalog on the home page.

Update Course

Users are able to edit any course that currently exists in the course catalog.

When the user clicks the 'Edit' button, they are sent to this page where it displays all the current information about the course. If the user must change anything they can do it here and click 'update course' and it will update the course.

Delete Course

Users are also able to delete any of the currently listed courses.

If the user wishes to remove a course, they just click the 'delete' button and that will bring them to this page.

Course Description

Each course has a more detailed description that shows a course description as well as credit hours.

Under each course that is listed in the catalog, the user can click the 'course description' link that will show them further details about the course that they clicked on.
